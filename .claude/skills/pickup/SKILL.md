快速接手项目，了解当前开发状态：

1. **阅读交接文档**
   - 阅读 @.claude/HANDOFF.md 了解：
     - 最近 session 完成的工作
     - 待实现功能
     - 已知问题

2. **了解项目结构**
   - 阅读 @.claude/CLAUDE.md 了解：
     - 项目概述和核心概念
     - 目录结构
     - 常用命令

3. **检查待办事项**
   - 查看 `.claude/specs/` 目录是否有未实现的 spec
   - 如有，阅读并理解需求

4. **确认当前状态**
   - 运行 `git status` 查看是否有未提交的变更
   - 运行 `git log --oneline -5` 查看最近提交

5. **准备就绪**
   - 总结当前项目状态
   - 列出可以继续的工作
   - 询问用户本次 session 的目标

6. **补充**
   - 当前项目是本地玩具项目，不需要考虑向后兼容
